# Asset Archive

This directory is the only destination for assets proven unused. It is intentionally empty as of 2026-07-22: a repository scan found asset files but did not establish that any are unreferenced by runtime code, service-worker cache, Pages allowlist, tests, or documentation.

Before moving an asset here, create a dated manifest containing its original path, SHA-256, reference-search results, mover, reason, timestamp, and restore command. Do not archive generated assets solely because their generator is unavailable.

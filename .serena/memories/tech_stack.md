# Stack
- ES modules on Node 22.x; npm 10.9.2; static browser/PWA without an application framework.
- Browser gameplay modules: `app.js`, `defense-run-simulation.js`, `defense-catalog.js`, `campaign-state.js`, `battle-realtime-three.js`, `battle-visualizer.js`, `defense-audio.js`, `defense-telemetry.js`, `defense-cutscene.js`.
- Dev dependencies: Playwright 1.52.0 and agentation 1.1.0.
- GitHub Pages is the production delivery path; only workflow allowlisted committed runtime assets ship.

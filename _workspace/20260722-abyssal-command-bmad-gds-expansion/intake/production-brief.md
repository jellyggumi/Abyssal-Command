- **team_shape:** 15-role studio (3× base harness): director, core five roles, and ten specialist lanes for combat balance, PCG, encounters, economy, audio systems, sound, VFX, narrative, accessibility, and telemetry. Artifacts remain evidence-bound and do not change runtime rules.
- **engine:** static browser/PWA; ES modules, Canvas snapshot renderers, local-first persistence.
- **current_stage:** Stage 1 FIX inherited as evidence context only; this is a new concept-and-systems expansion run, not a claim that inherited gates passed.
- **operating_mode:** concept-validation. One job: turn surveyed combat feedback, progression/reward, PCG stages/waves, world linkage, and immersive presentation into a bounded original vertical-slice design contract.
- **next_public_beat:** an internally playable **Cinder Span Command Feedback vertical slice**: one 90-second loop with readable health/crit/cooldowns/VFX/sound, varied authored-seed map and wave grammar, explicit growth/reward choices, a small offline idle-return claim, and a stage-to-world narrative handoff.
- **source_packet:** current product contract, current Stage 1 evidence/artifacts, and source-linked public deep-research survey.
- **main_constraint:** preserve Abyssal canon, single deterministic rules authority, offline local persistence, automatic baseline combat, movement-first controls, and reduced-motion accessibility. No commerce, account, network, cloud sync, copied proprietary content, or gameplay-time provider API dependency. ElevenLabs may only be a credential-isolated, build-time generated-asset source after rights review.

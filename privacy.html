Privacy Policy

Abyssal Command Privacy

Abyssal Command is an offline, browser-local game. The app does not load an analytics SDK, create an account, set cookies, upload telemetry, or show a consent banner.

Data on this device

  • Campaign progress and settings stay in this browser's local storage.
  • Diagnostic gameplay records stay in the running page and are saved only when you choose to download a local JSON file.
  • The app does not send campaign, gameplay, diagnostic, or identity data to a server.
  • Clearing this site's browser data removes locally stored progress unless you exported a copy.

Static hosting

The static hosting provider may process ordinary HTTP requests under its own policy to deliver these files. Abyssal Command does not receive or store those host request logs.
